WEB 1.0

Se conoce como la web "estática" debido a que no hay interacción entre los usuarios con el contenido de los sitios, los usuarios finales son pasivos (solo tenían permitido la lectura). Existían personas encargadas de generar contenido de manera específica para cada tema con un solo tipo de público.Su tecnología principal estaba basada en: Flash, Javascript y HTML.
Se considera la forma más básica que existe de navegadores de solo texto. Su principal objetivo era difundir información
Las pautas de accesibilidad eran 14:

  1. Proporcione alternativas equivalentes para el contenido visual y auditivo.
  2. No se base sólo en el color.
  3. Utilice marcadores y hojas de estilo y hágalo apropiadamente.
  4. Identifique el idioma usado.
  5. Cree tablas que se transformen correctamente.
  6. Asegúrese de que las páginas que incorporen nuevas tecnologías se transformen correctamente.
  7. Asegure al usuario el control sobre los cambios de los contenidos tempo-dependientes.
  8. Asegure la accesibilidad directa de las interfaces incrustadas.
  9. Diseñe para la independencia del dispositivo.
  10. Utilice soluciones provisionales.
  11. Utilice las tecnologías y pautas W3C.
  12. Proporcione información de contexto y orientación.
  13. Proporcione mecanismos claros de navegación.
  14. Asegúrese de que los documentos sean claros y simples.



WEB 2.0

Comprende aquellos sitios que facilitan compartir información. Estos permiten a los usuarios interactuar entre sí además de colaborar como creadores de contenido. Los usuarios dejan de ser simples receptores pasivos para convertirse en usuarios activos, produciendo una modificación de las conductas, usos y competencias de los internautas. Algunos ejemplos de esta red son: los servicios de red social, wikis y blogs. Sus principales características son: ser dinámica, interactiva y principalmente de lectura-escritura.
La web 2.0 no es más que la colaboración y el intercambio ágil de información entre los usuarios de una comunidad o red social.
Con esta nueva tecnología los profesores comienzan a dejar trabajos colaborativos donde los alumnos participan basados en los contenidos aportados por el profesor.
La comunicación entre empresas y clientes, se hace por medio de chats que pueden contactarse en forma sincrónica solicitando información




WEB 3.0

El término web 3.0 apareció por primera vez en 2006 en un artículo de Zeldman, crítico de la web 2.0. Esta web fue operativa desde el 2010

Son aplicaciones web conectadas, a fin de enriquecer la experiencia de las personas; a esto agrega conocimiento del contexto en la web geoespacial, la autonomía respecto del navegador y la construcción de la web semántica. La web 3.0 es conocida como la web semántica porque utiliza de forma más eficiente de los datos: data web. Es inter-operativa y el usuario tiene el control para hacer los cambios que desee modificando directamente las bases de datos. La web semántica incluye metadatos semánticos u ontológicos (que describen los contenidos y las relaciones entre los datos) para que puedan ser rastreados por sistemas de procesamiento.
Ofrecer un flujo de información y de contenidos adaptados a nuestros gustos y preferencias. Las redes sociales pasarían a ser ahora, redes o comunidades de aprendizaje. Es una red que promueve el acceso de las personas a la información, sin importar del medio de acceso, la interacción sirve para lograr resultados, no solo el hecho de compartir información.




WEB 4.0

Es el próximo gran avance y se centrará en ofrecer un comportamiento más inteligente, más predictivo, de modo que podamos, con sólo realizar una afirmación o petición, poner en marcha un conjunto de acciones que tendrán como resultando aquello que pedimos o decimos.
La web 4.0 permite la computación cognitiva. A través de ordenadores se almacenan en la nube permitiendo, no sólo que cualquier dispositivo con conexión a internet nos ayude a aprovecharnos de ellos.
Empresas como Google, Microsoft o Facebook, entre otras, están desarrollando nuevos sistemas que gracias al Deep Learning y Machine Learning serán capaces de procesar información de forma similar a como lo haría el cerebro humano. Tenemos los avances que los asistentes de voz están logrando. Siri, Google Now o Cortana entienden cada vez de forma más precisa y correcta lo que les decimos o solicitamos.
La experiencia del usuario es elemental al diseñar o crear plataformas, productos y servicios. Se trata de una web activa.
