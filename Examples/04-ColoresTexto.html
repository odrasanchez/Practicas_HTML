Colores sobre el texto

Para agregar color a todo el texto de una página, agregamos el atributo text = "hex" a la etiqueta <body>, donde "hex" puede ser un color en hexadecimal o ingresar el texto de algún color definido (en inglés) como: blue, yellow, magenta, red, black, white.


Para agregar color a toda la página, agregamos el atributo bgcolor = "hex" a la etiqueta <body>, donde "hex" puede ser un color en hexadecimal o ingresar el texto de algún color definido (en inglés) como: blue, yellow, magenta, red, black, white.


Esta es la etiqueta que se utilizó en esta página: <body bgcolor="TEAL" text="#FFFFFF">
